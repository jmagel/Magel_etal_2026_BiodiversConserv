****

Data and R code accompanying:

**Impacts of sea level rise on the world's shorebirds**

Authors: Jennifer M.T. Magel, Scott Wilson, Tara G. Martin

****

**CODE**
* analysis.R – This R script contains the necessary code to calculate the basic statistics included in the paper (e.g., mean study length; number of studies for each stage of the annual cycle, species, flyway, etc.).
* figures.R – This R script contains the necessary code to produce Figures 2, 6, and S1. Please note that additional processing was performed in PowerPoint to produce the final version of Fig. 6.
